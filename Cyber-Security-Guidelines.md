# Cyber Security Guidelines

## Research notes

In a remote work environment, some common cyber security threats include phishing emails, suspicious links, weak passwords, stolen credentials, malware, unsafe Wi-Fi, and accidentally sharing information in the wrong place. Remote work also means people may use different networks and devices, so good security habits are important.

Some best practices for keeping devices and accounts secure are:

- use strong, unique passwords
- use a password manager where possible
- enable two-factor authentication
- keep devices and apps updated
- lock devices when stepping away
- avoid clicking suspicious links or attachments
- only use approved tools for work
- report anything suspicious early

Locking my computer when I am away from my desk is important because someone else could access work systems, messages, documents, or internal information. Even if the risk seems small, it is a simple habit that helps protect company and user data.

For phishing attempts or suspicious links, I should slow down and check before clicking. I can look at the sender, the URL, the wording, and whether the request feels unusual or urgent. If I am unsure, I should not interact with it and should report it through the correct process.

A strong password should be long, unique, and hard to guess. Using a password manager helps because I do not need to reuse passwords or create weak ones that are easy to remember. It also makes it easier to use different passwords for different accounts.

Two-factor authentication is important because it adds another layer of protection if a password is stolen. I should enable 2FA on work accounts and any important personal accounts where it is available.

## Reflection

Some security measures I currently follow include using strong passwords, being careful with suspicious links, keeping my device locked when not in use, and using 2FA where available.

One area I can improve is being more consistent with reviewing password storage and making sure every important account has a unique password and 2FA enabled.

I can make secure behaviour a habit by building it into my normal workflow instead of treating it as something extra. For example, locking my screen when I stand up, checking links before opening them, and pausing before sharing files should become automatic habits.

To keep my passwords and accounts secure, I will:

- use strong and unique passwords
- use a password manager where appropriate
- enable 2FA wherever possible
- avoid reusing passwords across accounts
- review account security settings regularly
- report suspicious account activity quickly

If I suspected a security breach or suspicious activity on my account, I would report it as soon as possible. I would include what I noticed, when it happened, what account or system may be affected, and whether I clicked anything or entered any information. I would also change passwords or revoke sessions if instructed.

## New cyber security habit

One new habit I will follow at Focus Bear is to lock my laptop whenever I step away, even for a short time.

I will also be careful about where I store information, avoid using unapproved tools for work data, and report anything suspicious instead of assuming it is nothing.

## Key takeaway

Cyber security is not only about technical tools. It also depends on small daily habits like using strong passwords, enabling 2FA, locking devices, checking links, and reporting issues early.

As a Cyber Security Analyst Intern, I should model these habits consistently and be careful with both company and user information.
